# Formatierung

Verändern Sie den Aufruf von "printf" derart, sodass die beiden Variablen a und b nebeneinander ausgegeben werden. 
Die Variablen sollen bei der Ausgabe mit einem Leerzeichen getrennt werden.

```cpp
#include <stdio.h>

int main() {
  int a = 17;
  double b = 33.31;
  
  printf(a, b);
  
  return 0;
}
