# Array durchsuchen

Es werden vom Benutzer Ganzzahlen eingelesen und in einem Array abgelegt.
Geben Sie das größte und das kleineste Element auf der Konsole aus.

```cpp
#include <stdio.h>

int main(){
  int a[7];
  for (int i = 0; i < 7; i++) {
      scanf("%i", &a[i]);
  }

  // Ihr Quelltext ab hier
  
  //
  return 0;
}
