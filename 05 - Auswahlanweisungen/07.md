# Switch
Überführen Sie das gegebene *if-else*-Konstrukt in eine *switch-case*-Anweisung.

```cpp
#include <stdio.h>

int main() {
  double a, b;
  char c;
  scanf("%lf %lf %c", &a, &b, &c);

  if(c == '+') {
      printf("%f", a + b);
  } else if (c == '-') {
      printf("%f", a - b);
  } else if (c == '*') {
      printf("%f", a * b);
  } else if (c == '/') {
      printf("%f", a / b);
  } else
      printf("%c ist kein Operator", c);

  return 0;
}
```
