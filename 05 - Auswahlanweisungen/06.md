# Verbundene Bedingungen

Vereinfachen Sie die *if*-Anweisungen, sodass nur eine einzige *if*-Anweisung übrig bleibt.
Achten Sie darauf die Funktionalität des Programms nicht zu verändern!

```cpp
#include <stdio.h>

int main() {
  int a, b, c;
  bool result = false;
  scanf("%i %i %i", &a, &b ,&c);
  
  if(a > 100) {
    if(2 * b < 30) {
      if (!c) {
        result = true;
      }
    }
  }

  printf("%i", result);
  return 0;
}
```
