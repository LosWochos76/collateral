# Schleifen übersetzen

Wandeln Sie die gegebenen *while*-Schleifen in gleichartige *for*-Schleifen um.

```cpp
#include <stdio.h>

int main() {
  int zaehler1 = 10;
  while (zaehler1 >= 1) {
      printf("%i ", zaehler1);
      zaehler1--;
  }

  int zaehler2 = 5;
  while (zaehler2 <= 60){
      printf("%i ", zaehler2);
      zaehler2 = zaehler2 + 5;
  }
  
  return 0;
}
